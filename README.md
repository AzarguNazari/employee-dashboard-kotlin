# Employee Dashboard
A spring boot backend application with a good practice of REST.

## Technologies
- Kotlin/JDK8
- Spring boot
- Intellij
- JPA

## To run locally:
- `./gradlew bootRun`

## To run in dev mode:
- `docker-compose up -d`
- `./gradlew bootRun`
